Теги HTML и CSS

HTML-теги — основа языка HTML. Теги используются для разграничения начала и конца элементов в разметке. ... Каждый HTML-элемент обозначается начальным (открывающим) и конечным (закрывающим) тегом. Открывающий и закрывающий теги содержат имя тега.

  • P-Обычный текст
  • B-Жирный текст
  • Link-Подключение css Файла,Библеотек,и т.д
  • H1 H2 H3 H4 H5 H6-Заголовки
  • BUTTON-Кнопки
  • Script-Скрипты
  • Title-Заголовок сайта
  • IMG-Этот тег позволяет вставить:Гиф,картинку,и т.п.
  • Div-это преднозначен для выделение врагмента документа
  • Остальные теги вы можете найти по сыллке.
